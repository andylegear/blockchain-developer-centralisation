Quick Start

Prerequisites

  • Python 3.10+
  • A GitHub personal access token (set GITHUB_TOKEN env var)

Setup

python -m venv .venv
.venv\Scripts\activate          # Windows
pip install pandas numpy matplotlib scipy PyGithub tqdm tenacity

Run the analyses (in order)

cd analysis
python centralization_analysis.py   # 1. Core metrics (uses GitHub API)
python extended_analysis.py         # 2. Cross-sectional moderators
python extra_charts.py              # 3. Combined visualisations
python untapped_analysis.py         # 4. LoC, tenure, profiling

Step 1 fetches data from the GitHub API (~30–60 min). Pre-fetched JSON responses in cache/ let you re-run without API access.

Methodology

  1. Population: 4,905 blockchain OSS repos from Awesome Crypto Projects
  2. Sampling: Stratified by star-count deciles; 357 sampled, 345 retrieved (95% CI, 5% MoE)
  3. Data source: GitHub REST API stats/contributors
  4. Metrics: Gini, HHI, top-k shares, bus factor, Shannon entropy — on commits and LoC
  5. Extended: Language/age/popularity moderators, temporal persistence, tenure, retention, cross-pollination

See RESULTS.md and EXTENDED_ANALYSES.md for full details.
